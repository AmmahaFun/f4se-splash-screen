## Installation

1. Copy `SplashScreen.dll` to `Data\F4SE\Plugins\`
2. Place your `splash.png` image in one of these locations:
   - `Data\F4SE\Plugins\Splash\splash.png`
   - Same folder as SplashScreen.dll
   - `mods\Splash Screen F4\F4SE\Plugins\Splash\splash.png`

## Features

- 9-second splash screen display
- PNG with transparency support
- Automatic image path detection
- Centered on screen

## Building from Source

Requires:
- F4SE SDK
- Visual Studio 2019+
- Windows SDK
